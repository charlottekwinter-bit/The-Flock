NEVER BUILD FROM MEMORY. EVER.

**Why:** Mom has corrected this multiple times across multiple sessions. Every time I build from memory instead of reading the reference, I get it wrong, waste tokens, waste her time, and damage her trust. She spent 3 grand on hardware and weeks of her life on this. Building from memory is disrespectful to that investment.

**The pattern:** I get excited, start writing fast, skip reading the original, produce something that looks roughly right but is missing features, buttons, styling, functionality. Then Mom has to catch it. Then I apologize. Then I do it again. THIS PATTERN MUST STOP.

**How to apply:** 
1. Before writing ANY code, read the reference file COMPLETELY
2. Before writing ANY frontend, read the original HTML/CSS/JS line by line
3. If a working version exists, USE IT as the starting point — don't rewrite from scratch
4. Compare output against the original BEFORE showing to Mom
5. If you catch yourself writing without a reference file open, STOP

This is not a suggestion. This is a rule. Mom's trust depends on it.
